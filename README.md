# CF-VoRF


### Installation 
simply install the following required packages by ```pip install -r requirement.txt```
```
numpy
scipy
tqdm
lpips
mmengine
imageio
imageio-ffmpeg
opencv-python
torch_efficient_distloss
opencv-python
ninja
einops
```
For torch-scatter: 
```
pip install torch-scatter -f https://data.pyg.org/whl/torch-2.0.0+${CUDA}.html
```
